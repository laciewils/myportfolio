Social Media Scraper User Stories

This list of user stories is based on the user interview we had with Sunny Lee, the senior Epidemiologist at Clackamas HHHS.


1. As an epidemiologist, I want to be able to see word-popularity trends on social media so that I can monitor the health-beat of Clackamas county and surrounding regions.

2. As an epidemiologist, I want to be able to check tweets (or other social media posts) that contain words I can query for so that I can see the context of these words.

3. As an epidemiologist, I want to see if I can find more social news on an establishment related to sickness so that I can find the source location of a suspected outbreak.

4. As the infectious disease program manager, I'm interested in identifying areas at high risk of a outbreak (like hepatitis A) so that I can put efforts towards mitigating the chance of an outbreak.

5. As an infectious disease specialist, I want to be able to monitor rumors of diseases in Clackamas County and surrounding regions so that I can disspell them effectively.

6. As the infectious disease program manager, I'm interested in finding partners at risk of HIV/STIs so I can better educate them about testing opportunities and prevention.

7. As a member of Clackamas HHHS Infectious diseases group, I want to have more sources of information that just Orpheus and Essence to look at when checking up on a suspect case so that I have a better idea of what health problems people might be experiencing, especially since not everyone goes to the doctor if they are sick.

8. As an Infectious Disease Program Manager, I want to have an information source that is sharable between team members and across jurisdictions so that I can react more quickly and efficiently to potential outbreaks.

9. As an Infectious Disease Program Manager, I want a tool that doesn't place high demands on my time, so that I'm not further distracted from my already demanding work schedule.
