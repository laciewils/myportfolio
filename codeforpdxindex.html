What is the Clackamas County HHS Social Media Scraper?

Overview.

The social media scraper will allow the employees of Clackamas County HHHS to input search terms and parameters such as location and time frame to search for tweets that reference the terms of interest. The scraper will save a desired search and its results, and also present the results visually in graphs and maps.

Problem.

The challenge is making the interface as simple as possible for the group using it. In this case, there is a very small target audience that we are starting with. We can directly talk to these people to get feedback on what seems intuitive to them. Although we can get direct feedback from the future users, we must have general ideas laid out to guide them so that they aren't overwhelmed with possibilities. Another challenge will be making the search functions as helpful and specific as possible while remaining within the realm of possibility of the Twitter API that we are pulling from.

Goal.

The goal is to create an interface that has all of the features that would help the employees to retrieve useful information while also being simple enough for the employees to have the motivation to keep using it. The goal here is function and simplicity over artistic design.

User Stories

This list of user stories is based on the user interview we had with Sunny Lee, the senior Epidemiologist at Clackamas HHHS. These user stories are boiled down to measurable acceptance criteria to serve as the basis for the user flows and wireframes to come.

Personas

Annie

Registered Nurse

Age: 27 // Location: Eugene, OR

Key User Points

  • 1. Needs a simple and useful resource
  • 2. Wants to combat misinformation on social media
  • 3. Determine whether things heard through non-clinical mediums are true
  • 4. Spread education based on the results of the scraper

“I have a limited knowledge of technology and our shared drive at work is already messy. I need something very simple and intuitive that can deliver useful information.”

Elena

Infectious Disease Program Manager

Age: 30 // Location: Boston, MA

Key User Points

  • 1. Wants the ability to determine geographical areas at high risk of diseases like Hepatitis A
  • 2. Wants to be able to use results to send prevention messages on social media
  • 3. Frustrated by a lack of cross-jurisdictional data and access to other relevant databases
  • 4. Finds it can be difficult to pull data from the team's own database

“My team is understaffed with high demands on their time. We need quick and easy ways to receive and pull data."
